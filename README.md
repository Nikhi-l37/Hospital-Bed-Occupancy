# Hospital Bed Occupancy (Project 62A)

A full-stack application for predicting hospital bed occupancy using React and FastAPI.

## Project Structure
- `frontend/`: React application (Vite)
- `backend/`: FastAPI Python server

## Setup

### Backend
```bash
cd backend
# Install dependencies (ensure you have a virtual environment)
pip install -r requirements.txt
# Run server
py main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
